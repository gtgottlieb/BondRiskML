This directory contains the programs and data for Cochrane and Piazzesi 
"Bond Risk Premia." 

The main program is fbregslatest.m. This is a matlab program. That program will call all the data and invoke all the other programs as needed. 

The program fbmc.m does the monte carlos. 

The program macro.m forecasts stock returns.
